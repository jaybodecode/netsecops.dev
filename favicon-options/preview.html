Once you've chosen your favorite option, run these commands:

# For example, to use Option 1 (Hexagonal Lock):
cp favicon-options/option1-hexagon.svg public/favicon.svg

# Then regenerate PNG versions:
npm run generate

# Or manually convert with ImageMagick (if installed):
convert public/favicon.svg -resize 16x16 public/favicon-16.png
convert public/favicon.svg -resize 32x32 public/favicon-32.png
convert public/favicon.svg -resize 48x48 public/favicon-48.png
